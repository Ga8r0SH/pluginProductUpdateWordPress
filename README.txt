Плагин для массового обновления продуктов WooCommerce с помощью пользовательских полей.

## Установка

1. Загрузите файлы плагина в каталог `/wp-content/plugins/mass-update-products`, либо установите плагин через экран плагинов в WordPress.
2. Активируйте плагин через экран плагинов в WordPress.

## Функции

- Добавление пользовательского поля к продуктам WooCommerce.
- Массовое обновление значения пользовательского поля для нескольких продуктов.

## Как использовать

### 1. Добавление пользовательского поля

После активации плагина, вы сможете добавлять пользовательские поля к продуктам WooCommerce через административную панель. Эти поля можно будет использовать для обновления информации о продуктах.

### 2. Массовое обновление

Для массового обновления значений пользовательских полей для нескольких продуктов, перейдите в раздел плагина в административной панели WordPress, выберите продукты, которые хотите обновить, и укажите новое значение для пользовательского поля.

## Примечания

- Плагин разработан для WooCommerce и требует его наличия для корректной работы.
- Для успешного обновления пользовательских полей необходимо, чтобы продукты были правильно настроены в WooCommerce.
